Firefox 新鮮事

  • 全新功能 今天發布的版本帶來許多的改進，幫助你搜尋更加聰明、快速。打得更少，尋找更多，使用Firefox 翻新的網址列：
    • 專為小螢幕筆電最佳化，提供專注、簡潔的搜尋體驗。
    • 選擇網址列時將會出現熱門網站。
    • 改善了搜尋字詞的可讀性，凸顯新的搜索關鍵字。
    • 搜尋建議加入 Firefox 常見問題的解決方式。
    • 在 Linux 平台上，點擊網址列與搜尋列的行為現在與其他桌面平台相同：單擊選擇全部，但不進行快速複製（Primary Selection），雙擊會選擇單詞，三擊則選取全部並進行快速複製。
  • 全新功能 Firefox 會在本地暫存所有 Mozilla 已知且信任的 Web PKI 憑證機構所簽發的憑證，將會改善設定錯誤的網頁伺服器的 HTTPS 相容性並提高安全性。
  • 全新功能 Firefox 在 Flatpak 上架了。透過 Flatpak 可讓您更輕鬆在 Linux 安裝使用 Firefox。
  • 全新功能 已為我們的 Windows 使用者整合了 Direct Composition 以改善效能，也對我們在使用 Intel 顯示卡的 Windows 10 筆記型電腦預載 Firefox 的工作有所助益。
  • 修正 各種安全性修正。
  • 企業版 將 security.osclientcerts.autoload 偏好設定設成 true ，即可啟用對本機作業系統憑證儲存空間的實驗性支援。
  • 企業版 可使用企業政策來排除要透過 DNS over HTTPS 的可信任供應商（TRR, Trusted Recursive Resolver）解析的網域。
  • 開發功能 請參閱 Mozilla Hacks 部落格上的 Firefox 75: 四月新氣象 一文。
  • 開發功能 在 <img> 標籤使用 loading 屬性來節省頻寬與減少瀏覽器記憶體使用量。預設值 eager 會讓圖片立即載入，而 lazy 則會將圖片延後進入 viewport 範圍內才載入。
  • 開發功能 主控台的即時求值功能，使開發者可以比以往更快找出與修正錯誤。只要無副作用（side-effect free）的表達式一輸入到網頁主控台中，就可以即時預覽結果。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
  • 也請閱讀 Mozilla Hacks 開發者部落格：Firefox 75: 四月新氣象。
